# 用户与认证

本域共 4 张表，覆盖 C 端用户账号、登录日志、邀请关系、IP 注册限制。

## u_user — 用户主表

**用途**：存储 C 端用户账号信息，包括基础资料、创作币余额、会员状态、用户类型等。管理端通过 `PlatformUser` 实体共享访问。

| 字段 | 类型 | 允许空 | 默认值 | 含义 |
| --- | --- | --- | --- | --- |
| id | BIGINT UNSIGNED | 否 | AUTO_INCREMENT | 主键 ID |
| biz_no | VARCHAR(64) | 否 | - | 用户对外暴露的业务编号（如 U000123），注册时生成，不可变 |
| nickname | VARCHAR(64) | 是 | NULL | 用户昵称，1-20 字符，service 层校验 |
| email | VARCHAR(128) | 否 | - | 登录邮箱，唯一 |
| password_hash | VARCHAR(256) | 否 | - | BCrypt 加密后的密码哈希，原始密码不存 |
| avatar_url | VARCHAR(512) | 是 | NULL | 头像 URL，空时 UI 回退到昵称首字母 |
| invite_code | VARCHAR(16) | 是 | NULL | 个人邀请码（6 位），注册时生成，唯一 |
| coin_balance | DECIMAL(19,4) | 否 | 0.0000 | 创作币余额（正为可用） |
| user_status | TINYINT UNSIGNED | 否 | 1 | 用户状态：0-禁用，1-正常 |
| user_type | TINYINT UNSIGNED | 否 | 1 | 用户类型：0-机器人，1-真实用户 |
| membership_expire_at | DATETIME | 是 | NULL | 会员到期时刻（到期日次日 00:00，NULL=非会员） |
| membership_plan | VARCHAR(32) | 是 | NULL | 会员套餐：monthly/quarterly/yearly 等，NULL=无套餐 |
| email_verified | TINYINT UNSIGNED | 否 | 0 | 邮箱是否验证：0-否，1-是；改邮箱成功后置 1 |
| tenant_id | BIGINT UNSIGNED | 否 | 0 | 租户 ID，当前统一为 0，预留多租户扩展 |
| is_deleted | TINYINT UNSIGNED | 否 | 0 | 逻辑删除标记：0-未删，1-已删；查询自动追加 is_deleted=0 |
| created_at | DATETIME(3) | 否 | CURRENT_TIMESTAMP(3) | 创建时间 |
| updated_at | DATETIME(3) | 否 | CURRENT_TIMESTAMP(3) ON UPDATE | 更新时间 |
| created_by | BIGINT UNSIGNED | 否 | 0 | 创建人 ID |
| updated_by | BIGINT UNSIGNED | 否 | 0 | 更新人 ID |

**索引**：
- PRIMARY KEY (id)
- UNIQUE KEY uk_u_user_biz_no (biz_no)
- UNIQUE KEY uk_u_user_email (email)
- UNIQUE KEY uk_u_user_invite_code (invite_code)
- KEY idx_u_user_type (user_type)
- KEY idx_user_membership_expire_at (membership_expire_at)
- KEY idx_user_membership_plan (membership_plan)

**说明**：
- user_status 取值：0=禁用，1=正常。禁用时 JwtAuthenticationFilter 仍能解析 token，但登录接口拒绝。
- user_type 取值：0=机器人，1=真实用户。
- membership_expire_at 存"到期日次日 00:00"（如 7/10 24:00 结束 → 2026-07-11 00:00:00），用于管理端到期提醒。
- membership_plan 与 u_user_membership 表（V1.0.0_020）解耦：u_user 上是冗余快照，便于管理端列表查询；会员状态真源在 u_user_membership。
- 改邮箱时需同时清空旧 email 验证码缓存。

## u_user_login_log — 用户登录日志表

**用途**：记录用户登录/注册行为流水，用于安全审计与失败排查。

| 字段 | 类型 | 允许空 | 默认值 | 含义 |
| --- | --- | --- | --- | --- |
| id | BIGINT UNSIGNED | 否 | AUTO_INCREMENT | 主键 ID |
| user_id | BIGINT UNSIGNED | 否 | 0 | 用户 ID，0 表示未登录（如登录失败时尚未识别用户） |
| login_type | TINYINT UNSIGNED | 否 | 1 | 类型：1-密码登录，2-注册登录 |
| client_ip | VARCHAR(45) | 是 | NULL | 客户端 IP（兼容 IPv6） |
| user_agent | VARCHAR(512) | 是 | NULL | User-Agent |
| login_status | TINYINT UNSIGNED | 否 | 1 | 状态：0-失败，1-成功 |
| fail_reason | VARCHAR(256) | 是 | NULL | 失败原因，成功时为 NULL |
| created_at | DATETIME(3) | 否 | CURRENT_TIMESTAMP(3) | 创建时间 |

**索引**：
- PRIMARY KEY (id)
- KEY idx_u_user_login_log_user_id (user_id)
- KEY idx_u_user_login_log_created_at (created_at)

**说明**：
- user_id 关联 u_user.id；user_id=0 表示登录失败且未识别出用户的场景（如邮箱不存在）。
- login_type 取值：1=密码登录，2=注册登录（注册成功同时记为一次登录）。
- login_status 取值：0=失败，1=成功；失败时 fail_reason 记录原因（密码错误、账号禁用等）。

## u_user_invite_relation — 邀请关系表

**用途**：记录邀请人与被邀请人的绑定关系，用于邀请奖励与拉新统计。

| 字段 | 类型 | 允许空 | 默认值 | 含义 |
| --- | --- | --- | --- | --- |
| id | BIGINT UNSIGNED | 否 | AUTO_INCREMENT | 主键 ID |
| inviter_id | BIGINT UNSIGNED | 否 | - | 邀请人用户 ID |
| invitee_id | BIGINT UNSIGNED | 否 | - | 被邀请人用户 ID，唯一（一个用户只能被一个邀请人绑定） |
| invite_code | VARCHAR(16) | 否 | - | 邀请时使用的邀请码（冗余自 u_user.invite_code） |
| source_type | TINYINT UNSIGNED | 否 | 2 | 来源：1-链接，2-手动填写 |
| effective_status | TINYINT UNSIGNED | 否 | 0 | 状态：0-待验证，1-有效，2-无效 |
| created_at | DATETIME(3) | 否 | CURRENT_TIMESTAMP(3) | 创建时间 |
| updated_at | DATETIME(3) | 否 | CURRENT_TIMESTAMP(3) ON UPDATE | 更新时间 |

**索引**：
- PRIMARY KEY (id)
- UNIQUE KEY uk_u_user_invite_relation_invitee_id (invitee_id)
- KEY idx_u_user_invite_relation_inviter_id (inviter_id)

**说明**：
- inviter_id、invitee_id 均关联 u_user.id。
- source_type 取值：1=链接（通过邀请链接自动带上邀请码），2=手动填写（注册时手输邀请码）。
- effective_status 取值：0=待验证（注册完成但邮箱未验证等），1=有效（达到奖励条件），2=无效（被判定作弊或注销）。
- invitee_id 唯一约束保证一个用户只能被绑定一次。

## u_ip_register_limit — IP 注册累计限制表

**用途**：按客户端 IP 统计累计成功注册数，用于防批量注册与风控封禁。

| 字段 | 类型 | 允许空 | 默认值 | 含义 |
| --- | --- | --- | --- | --- |
| id | BIGINT UNSIGNED | 否 | AUTO_INCREMENT | 主键 ID |
| client_ip | VARCHAR(45) | 否 | - | 客户端 IP（兼容 IPv6），唯一 |
| register_count | INT UNSIGNED | 否 | 0 | 累计成功注册数 |
| is_blocked | TINYINT UNSIGNED | 否 | 0 | 是否永久封禁：0-否，1-是 |
| created_at | DATETIME(3) | 否 | CURRENT_TIMESTAMP(3) | 创建时间 |
| updated_at | DATETIME(3) | 否 | CURRENT_TIMESTAMP(3) ON UPDATE | 更新时间 |

**索引**：
- PRIMARY KEY (id)
- UNIQUE KEY uk_u_ip_register_limit_client_ip (client_ip)

**说明**：
- 每个 IP 一行，注册成功时 register_count +1；超过阈值或人工判定后置 is_blocked=1。
- is_blocked 取值：0=否，1=是；被封禁的 IP 注册接口直接拒绝。
