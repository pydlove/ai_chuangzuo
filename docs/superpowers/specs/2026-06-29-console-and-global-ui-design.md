# 控制台与全局 UI 设计

## 背景

爱创作原型目前由多个独立 HTML 页面组成，页面之间通过顶部导航栏切换。随着功能增加（消息中心、会员、设置、帮助等），需要一个统一的工作区来聚合登录后才能使用的功能。同时，全站需要统一的 Footer、关于我们和昼夜模式切换。

## 目标

- 新增一个「控制台」页面，作为登录后用户的统一工作区。
- 控制台包含左侧边栏导航和顶部 Header，Header 聚合头像、会员、消息、教程、昼夜切换、反馈、关于我们。
- 首页 / 会员页保持宣传网站形式不变，但底部加上统一 Footer。
- 全站支持黑夜 / 白天模式切换，状态持久化。
- 全站可打开「关于我们」弹框。
- 保持纯前端实现，不引入后端。

## 总体架构

| 页面类型 | 代表页面 | 说明 |
|---|---|---|
| 宣传页面 | `index.html`、`pricing.html` | 面向未登录访客，保持营销内容，仅加 Footer |
| 控制台 | `console.html` | 登录后统一工作区，含侧边栏和 Header |
| 功能页 | `create.html`、`loading.html`、`preview.html`、`works.html` | 保留现有功能，从控制台侧边栏跳转进入 |
| 认证页 | `login.html`、`forgot.html` | 保持现有认证流程 |

## 控制台页面（`console.html`）

### 页面布局

```
+------------------------------------------+
| Header                                   |
| [头像][会员]   爱创作   [铃][教][昼][反][关] |
+----------+-------------------------------+
| 侧边栏    | 主内容区                       |
| 创作      |                               |
| 生成队列  |                               |
| 预览导出  |                               |
| 我的作品  |                               |
+----------+-------------------------------+
```

### 左侧边栏

固定宽度 200px，包含：

- **创作**：进入创作流程
- **生成队列**：查看生成中的任务
- **预览导出**：预览和导出文章
- **我的作品**：查看已生成的文章

当前选中项高亮，鼠标悬停有背景反馈。

### 顶部 Header

从左到右：

1. **头像**：点击下拉显示「个人中心」「退出登录」。
2. **会员**：点击弹出会员信息浮层。
3. **系统名称「爱创作」**：点击回到控制台首页。
4. **消息铃铛**：点击弹出消息浮层。
5. **教程**：点击弹出教程浮层。
6. **昼夜切换**：点击切换全站主题。
7. **反馈**：点击打开反馈入口。
8. **关于我们**：点击弹出关于我们弹框。

## 消息铃铛弹框

点击铃铛弹出下拉浮层，顶部三个 Tab：

- **公告**：产品公告、新功能上线、运营活动等。
- **站内信**：个人提醒，如会员到期、续费成功、系统通知。
- **历史记录**：生成文章的历史记录，点击可跳转到对应文章预览。

每条消息显示标题、摘要、时间。支持「全部已读」和「清空本类」。

## 教程弹框

点击「教程」弹出浮层，包含三个入口：

- **帮助 / 文档**：从基础到专业技巧的快速指南，助你充分利用爱创作的功能。
- **观看 B 站**：视频教程、定期直播和课程，适合学习者，直播中设有在线答疑时间。
- **加入微信交流群**：充满活力的作者网络，提供帮助的渠道，分享创作技巧、经验和最佳实践。

## 会员信息弹框

点击 Header 上的「会员」弹出，显示：

- 当前会员等级
- 会员到期时间
- 剩余 AI 生成额度
- 底部「立即续费」按钮，跳转 `pricing.html`

## 关于我们弹框

参考星月写作关于我们弹框，结构：

- 标题：「关于我们」
- Logo + 名称「爱创作」+ 标语
- 平台理念说明（2-3 段文字）
- 底部按钮：用户协议、隐私政策、关注微信、联系电话、举报
- 弹框底部：© 2026 爱创作 · All Rights Reserved

## 反馈入口

Header 右侧「反馈」按钮，点击后弹出轻量反馈弹框，包含：

- 反馈类型（功能建议 / 问题反馈 / 其他）
- 反馈内容文本框
- 提交按钮（原型阶段点击后提示「已收到反馈」即可）

## 黑夜 / 白天模式

- 切换按钮在 Header 右侧。
- 点击后切换 `html` 或 `body` 的 `data-theme="dark"` 属性。
- 通过 CSS 变量定义颜色：`--bg-color`、`--text-color`、`--border-color`、`--card-bg`、`--primary-color` 等。
- 浅色为默认，深色下背景变深、文字变浅、边框变暗。
- 状态持久化到 `localStorage`，刷新后保持。

## 全站 Footer

每个页面底部统一加一行：

```
© 2026 爱创作 · All Rights Reserved | 备案号：XXX | 关于我们
```

点击「关于我们」打开弹框。

## 数据持久化

| 数据 | localStorage key | 说明 |
|---|---|---|
| 主题模式 | `aichuangzuo_theme` | `'light'` / `'dark'` |
| 消息数据 | `aichuangzuo_notifications` | 沿用现有，按公告 / 站内信 / 历史记录分类 |
| 会员信息 | `aichuangzuo_membership` | 等级、到期时间、剩余额度 |
| 登录状态 | `isLoggedIn` | 复用现有 |

## 非目标

- 不实现真正的用户协议 / 隐私政策页面内容（弹框内保持占位文案即可）。
- 不实现真实的反馈提交后端。
- 不改造现有功能页为 SPA 或嵌入 iframe。
- 不实现微信扫码关注、拨打电话等真实交互（原型阶段用提示框模拟）。

## 与现有功能的关系

- 复用现有 `shared.js` 中的通知 API、生成队列、作品列表等逻辑。
- 复用 `settings.html` 中的通知设置开关。
- 现有功能页保持可用，控制台作为新增入口逐步聚合。

## 后续可扩展

- 在控制台首页增加数据看板（今日生成数、剩余额度、最近消息）。
- 将 `create.html`、`works.html` 等逐步嵌入控制台主内容区。
- 增加团队 / 工作空间切换。
