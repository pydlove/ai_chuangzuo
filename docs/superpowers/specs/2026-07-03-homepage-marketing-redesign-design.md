# 首页营销重构设计

**日期**: 2026-07-03
**状态**: 已确认，待实现
**关联文件**: `project/user/web/src/views/Home.vue`

---

## 1. 功能概述

对爱创作用户端首页（`/`）做一次**营销内容重构**，目标是：

1. 从「AI 写作工具」叙事升级为「自媒体账号资产 + 多元变现」叙事；
2. 将平台已有的多种收益玩法（创作币、邀请返利、排行榜奖金、跨平台自媒体收入申报）在首页集中呈现；
3. 强化「账号越久越值钱」「早一天起号早一天开始滚雪球」的资产复利话术；
4. 在保留手机端适配的前提下，扩充桌面端与移动端的板块数量与文案。

**叙事主线**: 激情抓眼 → 理性证据 → 激情收尾；全面立体覆盖效率、变现、复利三条价值线。

---

## 2. 设计决策

- **改造范围**: 仅修改 `project/user/web/src/views/Home.vue`，不新增路由、不改动其他页面。
- **改造方式**: 拆掉重排现有四块（数据/特色/步骤/CTA），并新增「收益玩法矩阵」板块。
- **品牌主色**: `#FF2442`，保留现有色阶与暗色主题。
- **文案口吻**: Hero 与终 CTA 用激情表述抓眼球；中段（数据、特色功能、收益玩法矩阵）用理性、证据式表述。
- **手机端**: 沿用 `2026-07-03-mobile-homepage-design` 既有规范，新增板块同步适配 768px 断点。
- **暗色主题**: 所有新增样式同步支持 `body[data-theme="dark"]`。

---

## 3. 整体架构

新版首页按以下顺序垂直排列：

1. 顶部导航（保留，含「玩法指南」入口）
2. **Hero 区（重写，激情）**
3. **数据区（升级为收益+规模双驱动）**
4. **特色功能 6 张卡（保留 4 张 + 新增 2 张，第 6 张采用差异化视觉）**
5. **收益玩法矩阵（新增板块，4 张玩法卡 + 灰底背景 + outline CTA）**
6. 三步开始（化简为「3 步起号」，只承担上手叙事）
7. 终 CTA（激情：「现在起号，3 个月后看复利」）
8. 底部 Footer（保留）

### 叙事漏斗

```
激情（Hero）
  ↓ 理性证据（数据 → 6 张卡 → 收益玩法矩阵 → 三步）
  ↓ 激情收尾（终 CTA）
```

## 4. 各板块详细设计

### 4.1 导航（保留）

- 不改动。链接：首页 / 会员 / 玩法指南 / 主题切换 / 开始创作。
- 手机端抽屉沿用现有实现。

### 4.2 Hero 区（重写，激情）

| 元素 | 文案/规格 |
|---|---|
| 徽章 | `AI 写作助手 · 多平台变现 · 账号长期增值` |
| 主标题 | `会增值的自媒体账号,从第一篇文章开始` |
| 副标题 | `3 分钟产出一篇能直接发的文章,平台内多重赚钱机制,让你的自媒体账号像滚雪球一样,越做越大、越来越值钱。` |
| 主 CTA | `立即开始创作` → `/login` |
| 次 CTA | `看看能赚多少钱` → `/guide` |
| 三个 Checkmarks | `单篇 3 分钟成稿` / `多平台一稿多发变现` / `账号越久越值钱` |
| 引导链接 | `不知道怎么变现?先看看玩法指南 →` → `/guide` |

- 视觉沿用现有「粉红渐变 → 白」背景。
- 字号、按钮样式与现有保持一致。

### 4.3 数据区（升级，理性）

4 个数据指标，叙事从「工具能力」升级到「创作者已赚到钱」：

| 序号 | 数字 | 标签 |
|---|---|---|
| ① | **¥ 360 万 +** | 累计为创作者带来收益 |
| ② | **12 万 +** | 累计注册账号 |
| ③ | **6 大主流** | 已覆盖变现平台 |
| ④ | **3 分钟** | 平均成稿时间 |

- 数字为占位值，开发时根据真实数据替换。
- 视觉沿用现有 `#f8f9fa` 灰底 + 4 列布局；移动端保持 `2×2` 网格。
- 主数字字号与现有保持一致（桌面 36px / 移动 28px）。

### 4.4 特色功能 6 张卡（重排 + 新增，理性）

按「工具能力 → 价值理念」排列：

| # | 名称 | 副文案 |
|---|---|---|
| ① | **3 分钟成稿** | 输入写作方向，AI 自动完成标题、结构、正文。告别 3 小时憋一篇文。 |
| ② | **一稿多发跨平台** | 一次创作，公众号、小红书、抖音、百家号、头条、知乎全部适配，一份内容赚 N 份收益。 |
| ③ | **爆款结构** | 内置高打开率标题、钩子开头、金句结尾。不用懂写作也能产出爆款。 |
| ④ | **导出即发布** | 生成后预览、微调、导出 Word。复制到任何平台直接发布，快速变现。 |
| ⑤ | **持续变现**（NEW） | 创作币奖励、邀请好友返利、月榜奖金、外部自媒体收入申报……不是写一篇赚一篇，是越写越能赚。 |
| ⑥ | **账号资产复利**（NEW） | 自媒体账号像滚雪球 —— 粉丝、内容沉淀、平台权重，会随时间持续累加。早一天起号，早一天开始滚雪球。 |

**视觉规范**:

- ①—⑤：浅红图标底 + SVG 图标，**沿用** 现有 `.feature-card` 样式。
- ⑥：「账号资产复利」使用**差异化视觉** —— 深色背景 + 微型静态增长曲线（用 SVG 绘制 3 段折线 + ↑ 标识），主色 `#FF2442`。
- 移动端沿用现有 1 列堆叠。
- 桌面端从原 4 列改为 **6 列**（`repeat(6, 1fr)`），但卡片内容可能过窄 → 视情况降级为桌面 3 列 × 2 行；具体在实施时按排版效果调整。

> **注**: 若 6 列布局导致卡片过窄，将 6 张改为桌面 3 列 × 2 行（移动端保持 1 列）。实施时按实际效果二选一。

### 4.5 收益玩法矩阵（NEW，理性证据）

| 元素 | 文案/规格 |
|---|---|
| 板块标题 | `4 种变现路径,边写边赚` |
| 板块副标题 | `平台内赚创作币 + 返利 + 奖金,平台外赚自媒体收入` |
| 卡片 ① | **创作币奖励**：完成任务、活动、上榜，1 元 = 1 创作币。抵扣会员购买、满 100 可提现到支付宝。 |
| 卡片 ② | **邀请好友返利**：邀请 3 人 → 3 天会员；好友首单 10% 返利。老带新，你赚会员天数和创作币。 |
| 卡片 ③ | **排行榜奖金**：创作币榜、自媒体收入榜，月榜 TOP10 各奖 100 创作币 —— 写得好就上榜。 |
| 卡片 ④ | **自媒体收入申报**：公众号、小红书、抖音、百家号、头条、知乎 收益申报，记录你的自媒体收入轨迹。 |
| 底部 CTA | `查看完整玩法 · 看看别人赚了多少 →` → `/guide` |

- **视觉**: 4 列等宽，浅红图标底，SVG 图标，沿用 `.feature-card` 样式。
- **背景**: `#f8f9fa`（与数据区呼应，区隔于上方纯白特色功能区）。
- **CTA 样式**: outline 风格（不抢终 CTA 主按钮），指向 `/guide`。
- 移动端沿用现有 1 列堆叠。

### 4.6 三步开始（化简，理性）

由于前面板块已经讲清「能赚多少钱 / 为什么值得做」，这里只承担「上手」叙事。

```
标题    「3 步起一个会增值的账号」
副标题  「1 分钟注册，3 分钟第一篇，写不动也能保持账号在涨」

① 注册账号      ② 输入主题      ③ AI 产出
  1 分钟           1 句话          3 分钟可发
  (免费)           (零门槛)        (一篇成品)
```

- 视觉沿用现有红色渐变背景步骤区，圆角卡片样式保持不变。
- 数字徽章样式、间距沿用现有实现。
- 移动端保持垂直排列。

### 4.7 终 CTA（激情收尾）

| 元素 | 文案 |
|---|---|
| 标题 | `现在起号,3 个月后看复利` |
| 副标题 | `内容慢慢写,账号先到位 —— 等你准备好赚钱时,雪球已经在滚。` |
| 主 CTA | `立即开始创作` → `/login` |
| 次 CTA | `查看玩法指南` → `/guide` |

- 视觉沿用现有 `.cta-section` 居中布局、白底。
- 主按钮沿用 `.hero-btn` 红色阴影样式；次按钮采用 outline 风格，与「收益玩法矩阵」底部 CTA 视觉一致。

### 4.8 底部 Footer

- 不改动。

## 6. 响应式规则

### 6.1 顶部导航
沿用 `2026-07-03-mobile-homepage-design` 既有规范，不在本设计重复。

- 移动：`1fr` 单列。

### 6.6 三步

## 8. 不做的范围

- 不新增路由或页面。
- 不改动其他页面（玩法指南、登录、会员、定价、控制台等）。
- 不改造后台或后端 API。
- 不引入第三方营销组件或库（不使用 Vue 组件库的新组件，保持纯手写）。
- 不引入外部图标字体（继续使用内联 SVG）。
- 不做 A/B 测试或多版本（仅出一版）。
- 不做账号资产增长曲线图表的动态数据（用静态 SVG 即可）。
- 不做排行榜真实数据抓取（首页仅在「收益玩法矩阵」底部以 CTA 引导跳转，不嵌入真实榜单）。
- 不修改 2026-07-03 移动端适配既有规范中已固化的部分（断点、抽屉行为等）。
